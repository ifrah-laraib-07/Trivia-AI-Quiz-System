# Trivia AI Quiz System

This is a console-based quiz application developed in C++ as an Object-Oriented Programming semester project.

The project allows users to attempt chapter-based quizzes while administrators can manage questions. It also includes an AI chatbot module to assist users during the quiz.

## Features

- User and Admin login
- Chapter-wise quizzes
- Multiple-choice questions
- Score calculation
- File handling
- AI chatbot
- Modular OOP design

## Built With

- C++
- Object-Oriented Programming
- File Handling

## Project Structure

```
include/    Header files
src/        Source files
Chapters/   Quiz data
```

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio.
3. Build and run the project.
