# 🎯 Trivia AI Quiz System

A console-based **Trivia AI Quiz System** developed in **C++** as an Object-Oriented Programming (OOP) semester project. The application provides an interactive quiz experience with role-based access for students and administrators, chapter-wise quizzes, score tracking, and an AI chatbot assistant.

---

## 📌 Project Overview

The Trivia AI Quiz System is designed to make learning more engaging through interactive quizzes. It demonstrates the practical implementation of Object-Oriented Programming concepts while providing a user-friendly quiz platform.

The system supports multiple users, allows administrators to manage quiz content, and stores data using file handling techniques.

---

## ✨ Features

- 👤 User registration and login
- 🔑 Administrator login and management
- 📚 Chapter-based quizzes
- 🤖 AI chatbot assistance
- 📝 Multiple-choice questions
- 📊 Score calculation and display
- 💾 File handling for data storage
- 🧩 Modular object-oriented design
